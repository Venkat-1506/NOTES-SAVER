# 📝 Notes Saver

A Python console-based application to create, read, and delete notes using file handling.

---

## 🚀 Features
- Create new notes
- Read existing notes
- Delete notes
- Stores each note as a separate text file
- Automatic folder creation

---

## 🛠️ Technologies Used
- Python
- File Handling
- OS Module
